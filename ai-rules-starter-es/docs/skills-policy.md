# Política De Skills

Las skills pueden ayudar a un agente a realizar una tarea estrecha, pero no reemplazan la evidencia.

Política mínima:

- Nombrar cualquier skill usada.
- Mantener auditables las salidas de skills.
- No permitir que una skill amplíe el alcance silenciosamente.
- Preferir instrucciones del repo sobre orientación genérica de skills.
