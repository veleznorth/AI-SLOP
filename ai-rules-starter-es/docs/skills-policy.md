# Política de Skills

Las skills pueden ayudar a un agente a ejecutar una tarea estrecha, pero no reemplazan la evidencia.

Política mínima:

- Nombrar cualquier skill usada.
- Mantener auditables los outputs de skills.
- No dejar que una skill expanda el alcance en silencio.
- Preferir instrucciones del repo sobre guía genérica de skills.
